# 🕹️ Math Pac-Man: Binomial Box Method 

Welcome to **Math Pac-Man: Binomial Box Method**! This interactive educational game combines the classic arcade fun of Pac-Man with high school algebra, specifically designed to help students master the multiplication of binomials using the "Box Method."

## 📖 Description

In this game, players navigate a maze, gobble up dots, and avoid ghosts—just like the classic arcade game. However, when Pac-Man eats a **Power Pellet**, the action pauses, and the player is presented with a binomial multiplication problem (e.g.(x + 3)(x - 2)). To defeat the ghosts and earn bonus points, the player must successfully set up and solve the problem using an interactive drag-and-drop Box Method interface. 

It is entirely self-contained within a single HTML file, making it incredibly easy to host, share with students, or play offline!

## ✨ Features

- **Classic Arcade Gameplay:** Smooth, grid-based movement with CSS/Canvas-animated Pac-Man and ghosts (Blinky, Pinky, Inky, and Clyde).
- **Interactive Math Problems:** Power pellets trigger dynamic math problems.
- **Drag-and-Drop Interface:** Players drag binomial terms into the correct grid positions to construct the "Box" before solving.
- **Step-by-Step Checking:** Validates the setup (box headers), inner box calculations, and the final simplified equation.
- **Math Formatting:** Uses [KaTeX](https://katex.org/) to beautifully render mathematical equations.
- **Retro Audio & Visuals:** Includes embedded retro sound effects/music and 8-bit typography via Google Fonts ("Press Start 2P").
- **Score & Lives Tracking:** Keeps track of points earned through pellets and correct math problems, complete with a game-over and victory state.

## 🎮 How to Play

1. **Move:** Use the **Arrow Keys** or **W, A, S, D** to navigate Pac-Man through the maze.
2. **Eat Dots:** Gobble up the small yellow dots to increase your score.
3. **Avoid Ghosts:** Do not let the colored ghosts catch you, or you will lose a life!
4. **Power Pellets:** Eat the large flashing power pellets to trigger a **Math Challenge**.
5. **Solve the Box Method:** - *Drag and Drop* the terms of the binomials from the term bank into the top and side drop zones of the 2x2 grid.
   - *Multiply* the rows and columns to fill in the four inner boxes.
   - *Simplify* and type the final quadratic equation in the form ax^2 + bx + c.
   - Submit your answer to gain invincibility and chomp the ghosts for massive bonus points!

## 🛠️ Technologies Used

- **HTML5 Canvas:** For rendering the game board, player, and enemies.
- **Vanilla JavaScript:** Game loop, collision detection, and drag-and-drop math logic.
- **CSS3:** Retro styling, animations, and responsive modal overlays.
- **KaTeX:** For rendering professional-grade math notation inline.
- **Google Fonts:** "Press Start 2P" and "Segoe UI" for typography.

## 🚀 Setup & Installation

No build tools, servers, or dependencies are required to run this game!

1. Download the `index.html` file.
2. Double-click the file to open it directly in any modern web browser (Chrome, Firefox, Safari, Edge).
3. *Note: An active internet connection is required upon first load to fetch the KaTeX math library and Google Fonts.*

## ✏️ Customization (For Teachers & Developers)

You can easily modify the game's difficulty or the types of problems generated by editing the JavaScript inside the HTML file:
- **Math Generation:** Search for the `generateMathProblem()` function to change the range of coefficients and constants generated for the binomials. 
- **Game Speed & Difficulty:** You can tweak variables like `pacSpeed`, `ghostSpeed`, and `frightenedDuration` to make the arcade portion easier or harder for your students.

## 🎓 Educational Value

This tool transforms drill-and-practice algebraic expansion into a high-stakes, engaging challenge. It reinforces:
* Distributing terms systematically.
* Keeping track of negative and positive signs.
* Combining like terms to form standard quadratic equations.

Enjoy the game, and happy factoring! 🍒👻🟡
